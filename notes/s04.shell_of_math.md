bash 中 执行数学运算。

* 整数的数学运算：

    ```
    var1=$[2+3] # 2+3==5

    var2=100
    var3=50
    var4=45
    
    ret=$[$var2*($var3-$var4)] # 100*(50-45)==500
    ```
* 浮点数的数学运算

    > 运用到`bc`命令。一个支持浮点计算的东西。具体可以通过搜索`linux bc`查询

    ```
    var1=$(echo "2+3"|bc)

    var2=$(echo "scale=2;1/3"|bc) # 执行 1/3 的运算，保留小数点2位

    ```
